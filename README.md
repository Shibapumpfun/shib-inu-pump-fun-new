# SHIB INU Pump Fun

**SHIB INU Pump Fun** is a simple static website dedicated to Shiba Inu enthusiasts. It showcases fun information about the SHIB INU token and its community, with a playful and colorful design.  

## Features
- Responsive landing page
- Hero section with introduction
- About section describing the platform
- Footer with copyright
- Easy to customize
